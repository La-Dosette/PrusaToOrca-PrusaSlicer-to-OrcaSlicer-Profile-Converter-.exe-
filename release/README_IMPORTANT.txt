PrusaToOrca v0.2.0

Use this build from the Community project folder.

Recommended flow:
1. Pick a PrusaSlicer config bundle .ini.
2. Run Preview safe import.
3. Check the risk score and Advanced report.
4. Generate the .orca_printer bundle.
5. Import the generated bundle in OrcaSlicer with File > Import > Import Config Bundle.

Safety notes:
- Preview does not write files.
- Existing OrcaSlicer preset files are not edited by PrusaToOrca.
- Keep Prefix ON unless you know exactly why you need it disabled.
